* 맵을 여러 개 두는 복잡한 방식 대신, 한 칸에 `[땅 속성 | 드래곤 속성]`이 동시에 표현되는 통합형 2차원 리스트 UI 방식을 채택함.

### 🎮 최종 확정된 게임 규칙 및 흐름

* **기본 설정:** 3x3 크기의 격자 지도, 10턴 제한, 남은 코인이 최종 점수가 됨.
* **턴 시작:** 2턴째부터 알이 부화하며 숨겨진 땅의 정체(`🔥`, `💧`, `🌱`)가 드러남. 이미 부화한 드래곤이 땅 속성과 일치하면 코인 보너스(+100), 다르면 소량의 코인(+20)을 획득함.
* **턴 중 활동:** 코인을 내고 알을 원하는 좌표에 배치하거나, 상점에서 산 이동권을 사용해 잘못 배치된 드래곤을 알맞은 서식지로 이동시킴.
* **턴 종료:** 토토의 상점이 열려 코인을 소모해 '드래곤 이동권(200 Gold)'을 구매할 수 있음.

def place_egg(current_map):
    """사용자에게 좌표를 입력받아 알 [ 🥚 ]을 배치하는 함수"""
    print("\n[🥚 알 배치하기]")

        row = int(input("배치할 행 번호(0~2)를 입력하세요: "))
        col = int(input("배치할 열 번호(0~2)를 입력하세요: "))
        
        # [과제] 입력받은 좌표가 올바른 범위(0~2)이고, 빈칸('[ ❓ ]')인지 확인 후 알 배치하기
        # 예외 상황 처리 조건문을 작성해야 합니다.

        print(f"-> ({row}, {col}) 자리에 알을 배치했습니다!")

        print("❌ 잘못된 입력입니다. 숫자를 입력해 주세요.")

    input("\n계속하려면 엔터를 누르세요...")


def hatch_and_reward(current_map, secret_map, current_gold):
    """새 턴 시작 시 알을 부화시키고, 적절성에 따라 코인을 정산해주는 핵심 함수 (미완성)"""
    round_income = 0
    
    # 2차원 리스트 전체를 돌며 검사
    for i in range(3):
        for j in range(3):
            # 알이 있다면 부화 처리
            if current_map[i][j] == '[ 🥚 ]':
                if secret_map[i][j] == 'F': current_map[i][j] = '[🔥|🔥🐉]'

                elif secret_map[i][j] == 'G': current_map[i][j] = '[🌱|🌱🐉]'
                print(f"🎉 ({i}, {j}) 좌표의 알이 새로운 드래곤으로 부화했습니다!")

    return current_gold + round_income

    # 사용자가 나갈 때까지 반복하는 while문 구조가 필요합니다.
