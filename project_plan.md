내가 만들 프로그램의 주제:
드래곤 서식지 경영 게임

## 2. 문제 상황

내 설명:주어진 알을 알맞은 서식지에 배치하여 보다 높은 수익성을 얻고자 하는 것이 목적인 게임


---

## 3. 입력-처리-출력 구조

### 입력

사용자가 입력하거나 프로그램이 사용할 데이터는 무엇인가?

- place_egg(current_map): 사용자가 원하는 위치에 알([ 🥚 ])을 새롭게 배치하는 입력 함수

move_dragon(current_map): 이미 부화한 드래곤의 '현재 위치'와 '이동할 위치'를 각각 입력받아 자리를 바꾸는 입력 함수
- 
- 2차원 리스트를 사용해 서식지 현황을 직관적으로 드러내고자 함

### 처리

1. 알 배치 처리 (place_egg 함수 내부)
사용자가 입력한 행과 열 좌표를 검사하고, 문제가 없으면 2차원 리스트에 반영하는 처리입니다.

처리 순서:

사용자에게 row(행), col(열) 숫자를 입력받는다.

[조건문] 만약 입력된 숫자가 0 미만이거나 2를 초과하면 ➡️ "범위를 벗어났다"고 경고하고 다시 입력받는다.

[조건문] 만약 game_map[row][col]의 값이 '[ ❓ ]'가 아니라면 ➡️ "이미 다른 것이 있다"고 경고하고 다시 입력받는다.

위 조건들을 모두 통과하면 ➡️ game_map[row][col] = '[ 🥚 ]'로 데이터를 업데이트한다.

2. 부화 및 코인 정산 처리 (hatch_and_reward 함수 내부)
새로운 턴이 시작될 때 맵 전체를 조사하여 알을 드래곤으로 바꾸고, 서식지가 알맞은지 판단하여 코인을 계산하는 가장 중요한 처리입니다. 이중 for문과 if문이 주력으로 사용됩니다.

처리 순서:

이번 턴에 벌어들일 코인 변수(round_income)를 0으로 초기화한다.

[이중 반복문] 3x3 지도의 모든 칸(i행, j열)을 하나씩 훑어본다.

알 부화 처리 (hatch_and_reward 함수 실행 시):

3x3 격자 지도를 돌다가 알('[ 🥚 ]')을 발견합니다.

[이중 반복문] 3x3 지도의 모든 칸을 검사하기 전에, 이번 턴 총 수입 변수(round_income)를 0으로 둡니다.

만약 해당 칸이 알('[ 🥚 ]') 상태라면:

진짜 땅 속성(land_type)을 확인하고, 드래곤 속성(dragon_type)을 랜덤으로 결정합니다.

[조건문] 만약 land_type == dragon_type (방금 부화했는데 일치함) 이라면:

➡️ 일치 수입(50코인) + 부화 보너스(100코인) = 총 150코인을 더해줍니다.

[조건문] 만약 두 속성이 다르다면:

➡️ 불일치 수입(10코인)만 더해줍니다.

칸의 상태를 [땅|드래곤] 모양으로 업데이트합니다.

만약 해당 칸이 이미 부화해 있던 드래곤 상태라면:

[조건문] 땅 기호와 드래곤 기호가 일치하면 (예: '[🔥|🔥🐉]') ➡️ 매 턴 수입 50코인을 더해줍니다.

[조건문] 땅 기호와 드래곤 기호가 다르면 (예: '[🔥|💧🐉]') ➡️ 매 턴 수입 10코인을 더해줍니다.

## 5. 함수 설계

1. 지도 및 상태 출력 함수
함수 이름: display_status(name, turn, current_gold, items, current_map)

역할: 현재 턴, 플레이어의 자원(코인, 아이템), 그리고 3x3 격자 지도를 콘솔 창에 시각적으로 그려줍니다.

매개변수 (받는 값): * name (문자열): 플레이어 이름

turn (정수): 현재 턴수 (1~10)

current_gold (정수): 현재 보유 코인

items (정수): 보유 중인 이동권 개수

current_map (2차원 리스트): 플레이어용 지도(game_map)

반환값 (돌려주는 값): 없음 (None) - 화면 출력만 수행

2. 알 배치 함수
함수 이름: place_egg(current_map)

역할: 사용자에게 행과 열 번호를 입력받아 유효성을 검사한 뒤, 빈칸([ ❓ ])인 경우에만 알([ 🥚 ])을 배치합니다.

매개변수: current_map (2차원 리스트) - 알을 배치할 플레이어용 지도

반환값: 없음 (None) - 매개변수로 받은 리스트의 내부 값을 직접 수정

3. 부화 및 코인 정산 함수 (핵심)
함수 이름: hatch_and_reward(current_map, secret_map, current_gold)

역할: 새 턴 시작 시 알을 랜덤 속성 드래곤으로 부화시키고, 기존 드래곤들을 포함하여 서식지 일치 여부에 따라 코인을 계산해 합산합니다.

매개변수:

current_map (2차원 리스트): 플레이어용 지도

secret_map (2차원 리스트): 정답 지도(hidden_map)

current_gold (정수): 정산 전 보유 코인

반환값: updated_gold (정수) - 이번 턴에 번 코인이 더해진 새로운 최종 코인 값

4. 드래곤 서식지 이동 함수
함수 이름: move_dragon(current_map)

역할: 드래곤 이동권 아이템을 사용했을 때 실행됩니다. 옮길 드래곤의 원래 좌표와 새로 이동할 좌표를 입력받아 두 칸의 데이터를 서로 교환(Swap)합니다.

매개변수: current_map (2차원 리스트) - 드래곤을 이동시킬 지도

반환값: 없음 (None)

5. 아이템 상점 함수
함수 이름: open_shop(current_gold, items)

역할: 턴 종료 후 호출되어 플레이어가 코인을 소모해 '드래곤 이동권'을 구매할 수 있는 메뉴를 반복해서 띄워줍니다.

매개변수:

current_gold (정수): 현재 보유 코인

items (정수): 현재 보유 이동권 개수

반환값: (updated_gold, updated_items) (튜플 형태) - 아이템을 구매하고 남은 코인과 늘어난 아이템 개수를 동시에 반환
